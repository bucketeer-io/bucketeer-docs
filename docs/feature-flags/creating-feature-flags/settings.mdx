## Clone Flag

Creates a copy of the flag with the same targeting configuration and on/off state. You select the destination environment where the cloned flag will be created.

Bulk duplicate flag lets you copy a flag from one environment to several environments in a single step. Choose the flag you want to duplicate as the source, then select one or more destination environments. The system creates a copy of the flag in each selected environment, preserving the flag’s configuration while generating new variation IDs for the target environments. If a flag already exists in a destination environment, that environment is skipped and a status indicating the existing flag is returned.

## Archive Flag

Archives the flag in the current environment. Archived flags only appear when the Archived filter is selected in the flag list.
