This section presents the necessary information to use Bucketeer integration solutions. The integration solution on Bucketeer enables you to notify users systems or to receive notifications when some flag is created or updated on the dashboard.

This section provides essential information for utilizing Bucketeer integration solutions. These solutions enable you to notify user systems or receive notifications when a flag is created or updated on the dashboard.

## Pushes

[Push](/integration/pushes) are real-time updates to ensure the user application stays up to date with flag modifications or updates. Bucketeer uses Firebase Cloud Messaging (FCM) to send silent push notifications automatically.

## Notifications

Bucketeer provide dashboard [Notifications](/integration/notifications) to alert you and your team regarding changes or events on the dashboard.
