The Settings page allows you to configure your organization's basic information, including the organization name, URL code, description, and owner email.

## Accessing Organization Settings

1. Click the gear icon at the bottom of the left sidebar
2. Select **Settings** from the General section

<CenteredImg
  imgURL="img/organization-settings/settings-menu.png"
  alt="Organization settings form"
  wSize="100%"
/>

## Updating Settings

To update your organization settings:

1. Navigate to **Settings** from the settings menu
2. Modify the fields you want to update (Name, Description, or Owner Email)
3. Click the **Submit** button

:::info Permission Required
Only users with **Organization Admin** or **Organization Owner** roles can modify organization settings.
:::

:::caution URL Code Cannot Be Changed
The URL Code is a permanent identifier that cannot be modified after organization creation.
:::
